# Prompts para nuevo logo Puraqua

## Brief creativo

| Aspecto | Decisión |
|---|---|
| Tipo de logo | Combination mark (ícono + wordmark) |
| Ícono | Explorar 3 direcciones: gota, onda, gota+filtro |
| Estilo | Flat minimalista, colores sólidos |
| Forma | Circular |
| Texto | Solo "PURAQUA" — tagline aparte |
| Paleta | Navy #1a3a5c + cyan #00c8e0 |
| Tipografía | Sans-serif bold geométrica |

## Cómo usar este archivo

1. **Ronda 1:** Corré los Paso 1 de cada dirección en las 3 herramientas (9 prompts de ícono)
2. **Ronda 2:** Para los íconos que más te gusten, corré el Paso 2 correspondiente (agregar wordmark)
3. **Ronda 3:** Iterá sobre el ganador ajustando detalles

> Usá **Ideogram** y **Kittl** para explorar rápido. Usá **Recraft** para el resultado final en SVG.

## Guía de vectorización (si el logo viene de Ideogram)

Ideogram genera PNG, no vector. Si el logo ganador sale de ahí:

1. **Vectorizer.ai** (https://vectorizer.ai) — Mejor calidad AI, gratis con créditos
2. **Recraft Vectorizer** (https://recraft.ai/ai-image-vectorizer) — Rápido, profesional
3. **Adobe Express** (https://adobe.com/express) — Gratis, integra con Adobe

> Si el logo sale de **Recraft** o **Kittl**, ya es SVG. Exportar directo.

## Validación antes de usar

- [ ] **32px**: ¿Se entiende como favicon?
- [ ] **5 segundos**: ¿Alguien puede describirlo de memoria?
- [ ] **Contexto web**: ¿Funciona en navbar, footer, og:image?
- [ ] **Contexto redes**: ¿Funciona como avatar circular en WhatsApp/Instagram?
- [ ] **Fondo oscuro**: ¿La versión blanca se ve bien?
- [ ] **Impresión**: ¿Se ve bien en blanco y negro?
